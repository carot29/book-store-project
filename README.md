# book-store-code-for-a-client-project

our program helps organize books for an imaginary book store. You can search for a book by title, add new books, and see only low stock books. More will be added eventually!

instructions:
1: press Get Started.
2: choose one of the 3 buttons.
3: For add a book, input the title of the book, genre of the book, author of the book, and how many of the books in stock.
4: for the search  bar, simply input what you are looking for and you will see your results down below
5: click the back to menu button at any time to get back to the home screen
WARNING: if you input nothing into the add book inputs and press add, it will reload the page

the program can:
- Add books to the inventory
- search for books in the inventory
- show low stock

known limitations:
- cannot add stock to a book yet
- cannot delete a book from the list yet
- cannot filter alphabetically yet
